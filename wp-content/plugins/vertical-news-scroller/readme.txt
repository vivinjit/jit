== Description ==


Vertical News Scroller is a plugin for display vertical scrolling news for WordPress site. Admin can manage any number of news. Admin can add,edit delete news.

Simple and easy news scroller. Admin can have option for display text with news or just news title only. Admin can add a custom link to news. Pro version has category wise news and more features like support of thumbnail, news order, etc.

== Installation ==


This plugin is easy to install like other plug-ins of Wordpress as you need to just follow the below mentioned steps:

1. Admin can set any post or page as a news no need to add edit news manually.
2. Go to add or edit post or page and checkbox check add this post or page as news.

1. upload vertical-news-scroller folder to wp-Content/plugins folder.

2. Activate the plugin from Dashboard / Plugins window.

4. Now Plugin is Activated, Go to the Usage section to see how to use Vertical News Scroller.

### Usage ###

1.Use of Vertical News Scroller is easy after activating plugin go to Manage Scrolling News menu.

2.Add some news atleast two,three news for good,But It can be any number of news.

3.You can either use widget or shortcode for widget Go to Appearance-->Widgets and drag and drop "Vertical news scroll" widget to your desired location(like left,right,footer).

  for shortcode you can use shortcode [print_vertical_news_scroll maxitem="5" padding="10" add_link_to_title="1" show_content="1" delay="60" height="200px" width="100%"]

4.Thats it your scrolling news will appear.
